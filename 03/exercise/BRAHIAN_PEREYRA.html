  1. Que es un sistema de control de versiones?

    [Es una herramienta de software que monitoriza y gestiona cambios en un sistema de archivos. Lo que permite a los desarrolladores llevar un historial de los cambios que han realizado en el sistema de archivos. Esto permite que los cambios sean revertidos en caso de que se produzca un error.]

  2. Cual es la diferencia entre git y github?

    Github es una plataforma para alojar nuestros repositorios, mientras que Git es un sistema de control de versiones.

  3. Cómo creo una rama en git?

    [git branch nombre_de_la_rama]

  4. Cómo clono un repositorio?

    [git clone nombre_repositorio_remoto]
